# HOW TO RUN PROJECT

- Clone The repo and cd into your base folder

  `git clone https://github.com/DevCosmas/scissors_api.git`

  `cd <base folder>`

- Run "npm install" to install all the dependencies

  `npm install`

- npm start on the terminal to start the application

  `npm start`
